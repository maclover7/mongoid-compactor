# Mongoid::Compactor

A DSL for only reurning select JSON values. Designed to be the
ActiveModel::Serializers of Mongo.

# Quick Start

Add `mongoid-compactor` to your Gemfile:

```ruby
gem 'mongoid-compactor'
```

Stay tuned for more information.
